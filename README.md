# Similar Framework (Frontend + Backend)

## Frontend
- Path: `frontend`
- Stack: React + Vite + Monaco Editor + Socket.IO Client + face-api.js

## Backend
- Path: `backend`
- Stack: Express + Socket.IO

## Run

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Open:
- Frontend: http://localhost:5173
- Backend health: http://localhost:5000/api/health
